# Open VSX Registry

The Open VSX Registry is an open-source marketplace for VS Code extensions, providing an alternative distribution channel for extensions compatible with VS Code forks such as VSCodium, Cursor, and other Eclipse Theia-based editors.

## Purpose

The registry addresses the need to distribute extensions to the broader VS Code ecosystem beyond Microsoft's official marketplace. Many open-source projects and organizations prefer Open VSX for:

- Freedom from Microsoft's marketplace terms
- Support for non-Microsoft VS Code distributions
- Open-source governance of the registry itself

## Gemini CLI Availability

The Gemini CLI Companion extension is published on Open VSX Registry at: [https://open-vsx.org/extension/google/gemini-cli-vscode-ide-companion](https://open-vsx.org/extension/google/gemini-cli-vscode-ide-companion)

This ensures Gemini CLI integration is available to users of:

- VSCodium
- Cursor
- Other Open VSX-compatible editors
