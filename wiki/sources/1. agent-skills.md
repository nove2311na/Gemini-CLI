# Agent Skills

This source documents the Agent Skills feature for [[gemini-cli]], based on the [agentskills.io](https://agentskills.io/) open standard. Agent Skills enable specialized expertise on-demand through self-contained directories that package instructions and assets into discoverable capabilities.

## Key Topics Covered

- **Core functionality**: Self-contained skill directories with instructions and assets
- **Progressive disclosure**: Only metadata loaded initially; full content disclosed upon activation
- **Discovery tiers**: Four-tier hierarchy with precedence ordering
- **Activation lifecycle**: Five-step flow from discovery through execution
- **Management**: Interactive session commands (`/skills`) and terminal utilities (`gemini skills`)
- **Scope**: User-level (global) and workspace-level (project-local, version-controlled) skills

## Relationship to Other Wiki Pages

- Extends [[extensions]] by providing on-demand specialized expertise
- Interacts with [[confirmation-prompts]] for user consent before activation
- Part of the broader feature set documented across the wiki
