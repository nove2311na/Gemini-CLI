# System Requirements

To ensure optimal performance and stability, your development environment must meet the following minimum and recommended requirements for running Gemini CLI.

## Software Requirements

- **Operating Systems:**
    - macOS 15+
    - Windows 11 24H2+
    - Ubuntu 20.04+
- **Runtime:** Node.js 20.0.0 or higher.
- **Shells:** Bash, Zsh, or PowerShell.
- **Connectivity:** Active internet connection required for API communication and updates.

## Hardware Requirements

Hardware needs vary depending on the complexity of your tasks and the size of your codebases.

### "Casual" Usage
Suitable for short sessions, common code edits, and basic queries.
- **RAM:** 4GB+

### "Power" Usage
Recommended for long sessions, working with large codebases, and deep context analysis.
- **RAM:** 16GB+

## Location Availability

Gemini CLI is available in locations supported by **Gemini Code Assist**. Users should verify their region's availability to ensure uninterrupted service.
