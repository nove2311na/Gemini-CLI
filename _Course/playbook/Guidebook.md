# Lộ Trình Học Gemini CLI Mastery

Chào mừng bạn đến với khóa học chinh phục Gemini CLI. Lộ trình này được thiết kế để đưa bạn từ terminal beginner lên chuyên gia AI terminal.

## Quy Tắc Chung
- Luôn dùng prefix `rtk` cho các lệnh shell (ví dụ: `rtk ls`, `rtk npm`).
- Kiểm tra `Lesson_Content.md` trước để nắm lý thuyết.
- Thực hành đầy đủ các bước trong `Hands_on_Lab.md`.

---

## 📚 Danh Sách Các Module

### [Module 1: Cài đặt & Môi trường](./Module_1_Introduction_Environment_Setup/)
- **Mục tiêu**: Cài đặt thành công và authenticate Google account.
- **Nội dung chính**: npm install, auth methods, model selection (Pro/Flash).

### [Module 2: Workflow Cơ Bản & Lệnh](./Module_2_Core_Workflows_Command_Mastery/)
- **Mục tiêu**: Thao tác file và chạy shell an toàn.
- **Nội dung chính**: File management (@ syntax), shell execution (! prefix), .geminiignore.

### [Module 3: Lập Kế Hoạch Nâng Cao](./Module_3_Advanced_Task_Planning_Steering/)
- **Mục tiêu**: Sử dụng Plan Mode để thiết kế tính năng phức tạp.
- **Nội dung chính**: Research-Strategy-Execution, Model Steering (Hints), GEMINI.md.

### [Module 4: Subagents & Skills Chuyên Biệt](./Module_4_Specialization_Subagents_Skills/)
- **Mục tiêu**: Ủy thác task cho chuyên gia và mở rộng khả năng CLI.
- **Nội dung chính**: @codebase_investigator, custom agents, SKILL.md.

### [Module 5: Tùy Biến & Bảo Mật](./Module_5_Customization_Security_Guardrails/)
- **Mục tiêu**: Tối ưu hóa workflow và thiết lập rào chắn an toàn.
- **Nội dung chính**: Custom TOML commands, settings hierarchy, Sandboxing, Policy Engine.

### [Module 6: Tự Động Hóa & Hệ Sinh Thái](./Module_6_Automation_Ecosystem/)
- **Mục tiêu**: Tích hợp công cụ ngoài và chạy automation không giao diện.
- **Nội dung chính**: MCP servers, Headless Mode (-p flag), OpenTelemetry observability.

## 🛠 Cách Học Hiệu Quả
1. **Đọc**: Nghiên cứu lý thuyết súc tích.
2. **Code**: Gõ lệnh trực tiếp, không copy-paste mù quáng.
3. **Mở rộng**: Thử thay đổi prompt hoặc thêm điều kiện mới vào lab.
