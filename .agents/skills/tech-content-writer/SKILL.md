# Tech Content Writer

## Description
Transforms dry, technical documentation, release notes, or changelogs into engaging, high-quality content formats such as blog posts, newsletters, cheat sheets, or social media updates. Use this skill when asked to write a blog post, social media update, or newsletter about Gemini CLI features.

## Instructions
1. **Understand the Target Audience**: Identify whether the content is for beginners, advanced power users, or a general tech audience.
2. **Hook the Reader**: Start with a compelling hook, summarizing the value proposition or the problem the new feature solves.
3. **Maintain Technical Accuracy**: Do not alter technical facts. Always verify commands, flags, and paths against the official documentation in `Clippings/`.
4. **Format for Engagement**: Use formatting (bolding, lists, code blocks with syntax highlighting, callouts) to make the text scannable and engaging.
5. **Call to Action**: End with a clear next step (e.g., "Try it out using `gemini run ...`" or "Read the full docs here").
