# Issue and PR automation

Gemini CLI uses automated tools to manage the contribution workflow.

### Automated Review Tool

You can run the review tool to detect common anti-patterns and testing issues:

```bash
./scripts/review.sh <PR_NUMBER> [model]
```

### Issue Commands

*   `/assign`: Add a comment with this text to self-assign an issue (max 3 issues, must be labeled "help wanted").
*   `/unassign`: Add a comment with this text to unassign yourself.
